# Annotating
a place to do proof-of-concept webdesigns for publishing classical, heavily annotated texts. 
With the permission of William Nienhauser I'm using his annotations of *The Grand Scribe's Records* as an example.

Examples uploaded:

- [Claude Code out of the box](/pien_chueh.html) see [online](https://psmedia.github.io/Annotating/pien_chueh.html)
- Moving from validated TEI to html via XSL see [online](https://psmedia.github.io/Annotating/pien_chueh_1.html)
